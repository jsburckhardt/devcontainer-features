## Description

Installs the Flutter SDK from the official Flutter repository. Flutter is Google's UI toolkit for building beautiful, natively compiled applications for mobile, web, and desktop from a single codebase.

The feature includes:
- Flutter SDK installed in `/opt/flutter`
- Dart SDK (included with Flutter)
- VS Code Flutter extension support
